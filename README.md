## 🧰 Technology Stack

- **Framework**: Next.js  
- **Language**: TypeScript  
- **Database**: MySQL (remote or cloud-hosted)  
- **Authentication**: NextAuth.js  
- **Location Services**: Google Maps & IP-based geolocation  
- **Styling**: CSS Modules, SCSS
